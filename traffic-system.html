Traffic Light Controller

Below is a short slide given from a presentation for an integrated chip design to implement a sequential state machine to run a traffic light system. Using Verilog and Aldec to complete prototyping and test benching of a system emphasized the importance of perfecting and creating efficiency in systems. Boolean and Karnaugh simplification, while defining states and simplifying the problem of a traffic light system gave significant understanding to the importance of simplifying problems while optimizing solutions.


The lower-level nature of this assignment as it related to the circuit and hardware components still holds value for mid-level and high-level development. While emphasizing the importance of efficient code if for nothing else, resource management, higher level languages receive benefit from problem simplification and solution optimization similarly. While the propagation and latency problems are unpresented in these upper-level solutions, knowing of their existence emphasizes further the importance of keeping optimal code.
